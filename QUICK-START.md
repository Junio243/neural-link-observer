# 🚀 QUICK START GUIDE

## Getting Started in 30 Seconds

### Step 1: Download
```bash
git clone https://github.com/Junio243/neural-link-observer.git
cd neural-link-observer
```

### Step 2: Open
Double-click `index.html` or open it in your browser:
- Drag `index.html` to your browser
- Or right-click → Open with → Browser
- Or use `python -m http.server` and visit `localhost:8000`

### Step 3: Explore
That's it! No installation, no build process needed!

## 🎮 Interactive Features

### Terminal Commands
Click in the terminal and try:

```
$ help          # See all commands
$ status        # System status
$ neural        # Neural activity
$ metrics       # Performance data
$ info          # System specs
$ scan          # Run scan
$ echo Hello    # Echo text
$ clear         # Clear output
```

### Neural Graph
- Real-time animated waveform
- Shows current, average, and peak values
- Simulates AI neural activity patterns

### Matrix Feed
- Infinite scrolling log stream
- Color-coded events (green/magenta/red)
- AI/ML themed messages

### Metrics Panel
- CPU, Memory, Latency, Throughput
- Real-time animated bars
- Updates every 1.5 seconds

## 🐛 Troubleshooting

### Nothing appears?
- Check browser console (F12) for errors
- Make sure JavaScript is enabled
- Try a different browser
- Check all 3 files (HTML, CSS, JS) are in same folder

### Graphs not animating?
- Check CPU usage isn't maxed
- Try different browser
- Clear browser cache

### Commands not working?
- Make sure to press Enter
- Check spelling (case-insensitive)
- Try `help` to see available commands
